## [CentOS](http://centos.org) v6.8

* x86_64
* base install
* has VBoxGuestAdditions
* [Packer template](https://github.com/maier/packer-templates/)

## Use

```
#
# Create initial Vagrantfile
#

vagrant init maier/centos-6.8-x86_64

#
# Customize
#

vi Vagrantfile

#
# Start the box
#

vagrant up
```
